Custom Widgets Documentation
============================

This document provides a summary of the custom widgets used in the project, explaining their purpose and usage.

1. DynamicFormBuilder
---------------------
- Purpose: Automatically generates a complex Flutter form based on a JSON-like configuration (List of Field models).
- Usage: Used in the 'Add Post' and 'Project Bidding' screens to dynamically render fields sent from the backend.
- Key Features: Supports validation, conditional visibility, and various field types.
- File: lib/widget/form_widgets/dynamic_form_builder.dart

2. CustomButton
---------------
- Purpose: A stylized button widget with consistent app theme.
- Usage: Used for primary actions like 'Submit', 'Login', etc.
- File: lib/widget/form_widgets/app_button.dart

3. CustomCheckbox
-----------------
- Purpose: A single checkbox with a label.
- Usage: Used for 'Terms and Conditions' or simple boolean selections.
- File: lib/widget/form_widgets/app_checkbox.dart

4. CustomCheckboxGroup
----------------------
- Purpose: A group of checkboxes allowing multiple selections.
- Usage: Used for selecting multiple tags, categories, or options in a form.
- File: lib/widget/form_widgets/custom_checkbox_group.dart

5. CustomDatePickerForm
-----------------------
- Purpose: A text field that opens a date picker dialog.
- Usage: Used for selecting a single date (e.g., Anniversary, Birth Date).
- File: lib/widget/form_widgets/app_date_picker.dart

6. CustomDateTimePicker
-----------------------
- Purpose: Allows picking both date and time.
- Usage: Used for scheduling tasks or setting deadlines in forms.
- File: lib/widget/form_widgets/custom_date_time.dart

7. CustomDateRangePicker
------------------------
- Purpose: Allows selecting a start and end date.
- Usage: Used for filtering results by date range or selecting duration.
- File: lib/widget/form_widgets/app_date_range.dart & custom_date_time.dart

8. CustomToggle
---------------
- Purpose: A stylized switch/toggle button.
- Usage: Used for binary settings (e.g., Enable/Disable Notifications).
- File: lib/widget/form_widgets/custom_toggle.dart

9. CustomDatePicker
-------------------
- Purpose: A standalone date picker component.
- Usage: Embedded in other complex date selection widgets.
- File: lib/widget/form_widgets/custom_date_time.dart

10. CustomDateTimeRangePicker
-----------------------------
- Purpose: Picks a range of dates and times.
- Usage: Used for precise duration scheduling.
- File: lib/widget/form_widgets/custom_date_time.dart

11. DateTimeRangeResult
-----------------------
- Purpose: A helper model to store results from date range pickers.
- File: lib/widget/form_widgets/custom_date_time.dart

12. CustomDropdown
------------------
- Purpose: A stylized dropdown/select field.
- Usage: Used for selecting one option from a fixed list (e.g., Gender, City).
- File: lib/widget/form_widgets/app_dropdown.dart & custom_dropdown.dart

13. CustomRadioButton
---------------------
- Purpose: A single radio button component.
- Usage: Used as a building block for groups.
- File: lib/widget/form_widgets/app_radiobutton.dart

14. CustomRadioGroup
--------------------
- Purpose: A group of radio buttons for single selection.
- Usage: Used for selecting one option among several (e.g., Payment Method).
- File: lib/widget/form_widgets/custom_radio_group.dart

15. CustomTextfield
-------------------
- Purpose: The standard text entry field for the app.
- Usage: General inputs like Name, Email, Title.
- File: lib/widget/form_widgets/app_textfield.dart

16. CustomMobileNumberTextField
-------------------------------
- Purpose: A complex text field for phone numbers with country code selection.
- Usage: Used in registration and profile update screens.
- File: lib/widget/form_widgets/custom_mobile_number_textfield.dart

17. CustomTextarea
------------------
- Purpose: A multi-line text input field.
- Usage: Used for project descriptions, reviews, or comments.
- File: lib/widget/form_widgets/custom_textarea.dart

18. CustomTextfieldForAddress
-----------------------------
- Purpose: A specialized text field that integrates with map/location services.
- Usage: Used for picking addresses in forms.
- File: lib/widget/form_widgets/custom_textfield_address.dart

19. LocationPickerScreen
------------------------
- Purpose: A full-screen map interface for selecting a location pin.
- File: lib/widget/form_widgets/location_picker.dart

20. CustomFilePicker
--------------------
- Purpose: Allows picking a single file (image, video, document).
- File: lib/widget/form_widgets/custom_file_picker.dart

21. CustomMultipleFilePicker
----------------------------
- Purpose: Allows selecting and managing multiple files at once.

- File: lib/widget/form_widgets/custom_multiple_file_picker.dart
